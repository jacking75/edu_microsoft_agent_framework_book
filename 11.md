## 11.1 MCP (Model Context Protocol) 개요

### MCP란?

MCP(Model Context Protocol)는 Anthropic이 제안한 개방형 표준으로, LLM 애플리케이션과 외부 데이터 소스/도구를 연결하는 방법을 표준화한다.

```
┌─────────────────────────────────────────────────────────┐
│                    MCP 아키텍처                          │
│                                                          │
│  ┌──────────────┐     MCP 프로토콜     ┌──────────────┐ │
│  │  MCP Client  │◄───────────────────►│  MCP Server  │ │
│  │  (Agent)     │                     │  (도구/데이터)│ │
│  └──────────────┘                     └──────────────┘ │
│                                                          │
│  MCP Server가 제공하는 것:                               │
│  - Resources: 파일, DB, API 데이터                      │
│  - Tools: 실행 가능한 함수                              │
│  - Prompts: 재사용 가능한 프롬프트 템플릿               │
└─────────────────────────────────────────────────────────┘
```

### MCP vs 일반 Tool

```
일반 Tool:
  Agent ──► Tool 함수 (동일 프로세스 내)

MCP Tool:
  Agent ──► MCP Client ──► MCP Server (별도 프로세스/서버)
             (표준 프로토콜)
```

**MCP의 장점:** 도구를 독립적으로 배포하고 여러 Agent에서 재사용할 수 있다. 표준 프로토콜이므로 다양한 LLM 프레임워크에서 동일한 MCP 서버를 사용할 수 있다.

## 11.2 로컬 MCP 서버 구현

### .NET으로 간단한 MCP 서버 구현

Microsoft Agent Framework는 MCP 서버를 직접 구현하거나 기존 MCP 서버와 통합할 수 있다. 여기서는 HTTP 기반 경량 MCP 서버를 구현한다.

```csharp
// McpServer/Program.cs
using System.Text.Json;

var builder = WebApplication.CreateBuilder(args);
var app = builder.Build();

// MCP Tools 정의
var tools = new[]
{
    new
    {
        name = "get_system_info",
        description = "현재 시스템 정보(OS, CPU, 메모리)를 반환한다.",
        inputSchema = new
        {
            type = "object",
            properties = new { },
            required = Array.Empty<string>()
        }
    },
    new
    {
        name = "calculate",
        description = "수학 계산을 수행한다.",
        inputSchema = new
        {
            type = "object",
            properties = new
            {
                expression = new { type = "string", description = "계산할 수식 (예: 2+3*4)" }
            },
            required = new[] { "expression" }
        }
    }
};

// MCP 엔드포인트: 도구 목록 반환
app.MapGet("/tools", () => new { tools });

// MCP 엔드포인트: 도구 실행
app.MapPost("/tools/call", async (HttpContext ctx) =>
{
    using var reader = new StreamReader(ctx.Request.Body);
    var body = await reader.ReadToEndAsync();
    var request = JsonDocument.Parse(body);

    var toolName = request.RootElement.GetProperty("name").GetString();
    var arguments = request.RootElement.TryGetProperty("arguments", out var args)
        ? args : default;

    var result = toolName switch
    {
        "get_system_info" => GetSystemInfo(),
        "calculate" => Calculate(arguments),
        _ => "알 수 없는 도구입니다."
    };

    return Results.Json(new { content = new[] { new { type = "text", text = result } } });
});

app.Run("http://localhost:5100");

static string GetSystemInfo()
{
    return $"""
        OS: {Environment.OSVersion}
        CPU 코어: {Environment.ProcessorCount}
        메모리: {GC.GetTotalMemory(false) / 1024 / 1024}MB 사용 중
        .NET: {Environment.Version}
        """;
}

static string Calculate(JsonElement arguments)
{
    try
    {
        var expression = arguments.GetProperty("expression").GetString() ?? "";
        // 안전한 계산: 기본 사칙연산만 허용
        var result = EvaluateSimple(expression);
        return $"{expression} = {result}";
    }
    catch
    {
        return "계산 오류: 올바른 수식을 입력해주세요.";
    }
}

static double EvaluateSimple(string expr)
{
    // DataTable을 이용한 간단한 수식 계산
    var dt = new System.Data.DataTable();
    var result = dt.Compute(expr, "");
    return Convert.ToDouble(result);
}
```

### MCP 서버를 호출하는 Tool Wrapper

Agent에서 MCP 서버를 호출하는 Tool Wrapper를 만든다.

```csharp
// McpClientTools.cs
public class McpClientTools
{
    private readonly HttpClient _http;
    private readonly string _serverUrl;

    public McpClientTools(string serverUrl = "http://localhost:5100")
    {
        _http = new HttpClient();
        _serverUrl = serverUrl;
    }

    /// <summary>현재 시스템 정보를 MCP 서버에서 가져온다.</summary>
    public async Task<string> GetSystemInfo()
    {
        return await CallMcpTool("get_system_info", new { });
    }

    /// <summary>MCP 서버를 통해 수학 계산을 수행한다.</summary>
    /// <param name="expression">계산할 수식 (예: 125 * 847)</param>
    public async Task<string> Calculate(string expression)
    {
        return await CallMcpTool("calculate", new { expression });
    }

    private async Task<string> CallMcpTool(string toolName, object arguments)
    {
        var payload = JsonSerializer.Serialize(new
        {
            name = toolName,
            arguments
        });

        var content = new StringContent(payload, System.Text.Encoding.UTF8, "application/json");
        var response = await _http.PostAsync($"{_serverUrl}/tools/call", content);
        var json = await response.Content.ReadAsStringAsync();

        var doc = JsonDocument.Parse(json);
        var contentArr = doc.RootElement.GetProperty("content");
        return contentArr[0].GetProperty("text").GetString() ?? "";
    }
}
```

### Agent에서 MCP Tool 사용

```csharp
using DotNetEnv;
using AgentFrameworkBook.Shared;
```

## 11.3 OpenRouter 고급 활용

### 모델 비교 및 선택 전략

OpenRouter는 다양한 LLM을 통합 API로 제공한다. 용도에 따라 최적의 모델을 선택하는 전략이 중요하다.

```
┌────────────────────────────────────────────────────────────┐
│                  모델 선택 가이드                          │
├──────────────────┬──────────────────┬──────────────────────┤
│ 용도             │ 추천 모델        │ 이유                 │
├──────────────────┼──────────────────┼──────────────────────┤
│ 빠른 응답       │ claude-haiku-4-5 │ 저비용, 빠름         │
│ 일반 작업       │ claude-sonnet-4-5│ 균형잡힌 성능        │
```

### 동적 모델 선택기 구현

```csharp
// ModelSelector.cs
public enum TaskComplexity
{
    Simple,    // 간단한 질답
    Medium,    // 일반 작업
    Complex,   // 복잡한 추론/분석
    Expert     // 전문가 수준
}

public class ModelSelector
{
    private static readonly Dictionary<TaskComplexity, string> OpenRouterModels = new()
    {
        [TaskComplexity.Simple]  = "anthropic/claude-haiku-4-5",
        [TaskComplexity.Medium]  = "anthropic/claude-sonnet-4-5",
        [TaskComplexity.Complex] = "anthropic/claude-opus-4",
        [TaskComplexity.Expert]  = "anthropic/claude-opus-4"
    };

    private static readonly Dictionary<TaskComplexity, string> PoeModels = new()
    {
        [TaskComplexity.Simple]  = "claude-haiku-4-20250514",
        [TaskComplexity.Medium]  = "claude-sonnet-4-20250514",
        [TaskComplexity.Complex] = "claude-opus-4-20250514",
        [TaskComplexity.Expert]  = "claude-opus-4-20250514"
    };

    public static TaskComplexity Analyze(string prompt)
    {
        var wordCount = prompt.Split(' ').Length;
        var hasCode = prompt.Contains("코드") || prompt.Contains("구현") || prompt.Contains("알고리즘");
        var hasAnalysis = prompt.Contains("분석") || prompt.Contains("비교") || prompt.Contains("전략");
        var hasMath = prompt.Contains("계산") || prompt.Contains("수식") || prompt.Contains("통계");

        if ((hasCode || hasAnalysis || hasMath) && wordCount > 30)
            return TaskComplexity.Complex;
        if (wordCount > 20 || hasCode || hasAnalysis)
            return TaskComplexity.Medium;
        return TaskComplexity.Simple;
    }

    public static string GetModel(TaskComplexity complexity, string provider = "openrouter")
    {
        return provider.ToLower() == "poe"
            ? PoeModels[complexity]
            : OpenRouterModels[complexity];
    }
}
```

### 동적 모델 선택 Agent

```csharp
using DotNetEnv;
using AgentFrameworkBook.Shared;

Env.Load();

var apiKey  = Environment.GetEnvironmentVariable("LLM_API_KEY")!;
var baseUrl = Environment.GetEnvironmentVariable("LLM_BASE_URL") ?? "https://openrouter.ai/api/v1";
var provider = baseUrl.Contains("poe") ? "poe" : "openrouter";

Console.WriteLine("🧠 동적 모델 선택 Agent\n");

var queries = new[]
{
    "안녕하세요",                                           // Simple
    "파이썬과 C#의 차이를 설명해줘",                       // Medium
    "분산 시스템에서 CAP 정리를 고려한 데이터베이스 설계 전략을 분석해줘"  // Complex
};

foreach (var query in queries)
{
    var complexity = ModelSelector.Analyze(query);
    var model = ModelSelector.GetModel(complexity, provider);

    Console.WriteLine($"❓ 질문: {query}");
    Console.WriteLine($"📊 복잡도: {complexity} → 모델: {model}");

    var agent = AIAgentBuilder
        .Create(apiKey, baseUrl, model)
        .Build("AdaptiveAgent", "전문적이고 정확한 어시스턴트다.");

    var result = await agent.RunAsync(query);
    Console.WriteLine($"🤖 {result.Text[..Math.Min(100, result.Text.Length)]}...\n");
}
```

## 11.4 모델 전환 전략

### Fallback 전략

기본 모델이 실패하거나 할당량을 초과한 경우 자동으로 대체 모델로 전환한다.

```csharp
// FallbackAgent.cs
public class FallbackAgentBuilder
{
    private readonly List<(string baseUrl, string model)> _providers = new();
    private readonly string _apiKey;

    public FallbackAgentBuilder(string apiKey) => _apiKey = apiKey;

    public FallbackAgentBuilder AddProvider(string baseUrl, string model)
    {
        _providers.Add((baseUrl, model));
        return this;
    }

    public FallbackAgentRunner Build(string name, string instructions)
        => new(_apiKey, _providers, name, instructions);
}

public class FallbackAgentRunner
{
    private readonly string _apiKey;
    private readonly List<(string baseUrl, string model)> _providers;
    private readonly string _name;
    private readonly string _instructions;

    public FallbackAgentRunner(
        string apiKey,
        List<(string baseUrl, string model)> providers,
        string name,
        string instructions)
    {
        _apiKey = apiKey;
        _providers = providers;
        _name = name;
        _instructions = instructions;
    }

    public async Task<string> RunAsync(string input)
    {
        Exception? lastError = null;

        foreach (var (baseUrl, model) in _providers)
        {
            try
            {
                Console.WriteLine($"  [시도] {model}");
                var agent = AIAgentBuilder
                    .Create(_apiKey, baseUrl, model)
                    .Build(_name, _instructions);

                var result = await agent.RunAsync(input);
                Console.WriteLine($"  [성공] {model}");
                return result.Text;
            }
            catch (Exception ex)
            {
                Console.WriteLine($"  [실패] {model}: {ex.Message}");
                lastError = ex;
                await Task.Delay(1000); // 잠시 대기 후 다음 시도
            }
        }

        throw new Exception($"모든 provider 실패: {lastError?.Message}");
    }
}
```

### Fallback 사용 예제

```csharp
using DotNetEnv;

Env.Load();

var apiKey = Environment.GetEnvironmentVariable("LLM_API_KEY")!;

// OpenRouter → Poe 순으로 Fallback
var agent = new FallbackAgentBuilder(apiKey)
    .AddProvider("https://openrouter.ai/api/v1", "anthropic/claude-sonnet-4-5")
    .AddProvider("https://openrouter.ai/api/v1", "openai/gpt-4o-mini")
    .AddProvider("https://api.poe.com/llm/v1",   "claude-haiku-4-20250514")
    .Build("FallbackAgent", "신뢰할 수 있는 어시스턴트다.");

Console.WriteLine("🔄 Fallback 전략 Agent\n");

var result = await agent.RunAsync("AI 에이전트의 장점을 3가지 말해줘");
Console.WriteLine($"✅ 결과: {result}");
```

### Provider별 비용 최적화

```csharp
// CostOptimizer.cs
public class CostOptimizer
{
    // 대략적인 비용 (1M 토큰당 USD, 2025년 기준)
    private static readonly Dictionary<string, decimal> InputCosts = new()
    {
        ["anthropic/claude-haiku-4-5"]   = 0.80m,
        ["anthropic/claude-sonnet-4-5"]  = 3.00m,
        ["anthropic/claude-opus-4"]      = 15.00m,
        ["openai/gpt-4o-mini"]           = 0.15m,
        ["openai/gpt-4o"]                = 2.50m,
        ["google/gemini-flash-1.5"]      = 0.075m,
    };

    public static string GetCheapestModel(int estimatedTokens, decimal budgetUsd)
    {
        var affordable = InputCosts
            .Where(kv => (kv.Value / 1_000_000m) * estimatedTokens <= budgetUsd)
            .OrderByDescending(kv => kv.Value) // 가장 비싼 (= 성능 좋은) 것 선택
            .FirstOrDefault();

        return affordable.Key ?? "anthropic/claude-haiku-4-5"; // 기본값
    }

    public static void PrintCostEstimate(string model, int inputTokens, int outputTokens)
    {
        if (!InputCosts.TryGetValue(model, out var inputCost)) return;

        var cost = (inputCost / 1_000_000m) * inputTokens;
        Console.WriteLine($"  💰 예상 비용: ${cost:F4} (입력 {inputTokens:N0} 토큰)");
    }
}
```

```bash
ollama pull phi3
ollama pull mistral

# 서버 실행 (기본적으로 자동 실행됨)
ollama serve
```

```csharp
// 동일한 질문에 대한 응답 캐싱
var cache = new Dictionary<string, string>();

async Task<string> AskWithCache(AIAgent agent, string question)
{
    var key = question.ToLower().Trim();
    if (cache.TryGetValue(key, out var cached))
    {
        Console.WriteLine("  [캐시 히트]");
        return cached;
    }

    var result = await agent.RunAsync(question);
    cache[key] = result.Text;
    return result.Text;
}
```

### 팁 5: 응답 품질 검증

```csharp
async Task<string> RunWithValidation(AIAgent agent, string input, Func<string, bool> validator)
{
    const int maxRetries = 3;

    for (int attempt = 1; attempt <= maxRetries; attempt++)
    {
        var result = await agent.RunAsync(
            attempt == 1 ? input : $"{input}\n\n(이전 응답이 올바르지 않았습니다. 다시 시도해주세요.)"
        );

        if (validator(result.Text))
            return result.Text;

        Console.WriteLine($"  [검증 실패 - {attempt}/{maxRetries}]");
    }

    throw new Exception("유효한 응답을 받지 못했습니다.");
}

// 사용 예: JSON 응답 검증
var jsonResult = await RunWithValidation(
    agent,
    "플레이어 목록을 JSON 배열로 반환해줘",
    response => response.TrimStart().StartsWith("[") && response.TrimEnd().EndsWith("]")
);
```
