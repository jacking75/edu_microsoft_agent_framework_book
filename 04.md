에이전트에게 실제 작업을 수행할 수 있는 능력을 부여한다. Tools(도구)를 추가하면 에이전트가 외부 함수를 호출하고, API를 사용하고, 데이터를 처리할 수 있다. `AIAgentBuilder`를 사용하여 Tool을 간편하게 등록하는 방법을 배운다.

## 4.1 Function Tools 개념

### Tool Calling의 동작 원리

```
사용자: "서울 날씨 알려줘"
    ↓
Agent → LLM: 요청 + 사용 가능한 도구 목록 전송
    ↓
LLM: 분석 → GetWeather 함수 필요 판단
    ↓
Agent → Tool: GetWeather("서울") 실행
    ↓
Tool → Agent: "맑음, 22도" 반환
    ↓
Agent → LLM: 도구 실행 결과 전송
    ↓
LLM: "서울은 맑고 22도입니다" 생성
    ↓
사용자에게 최종 답변 전달
```

**중요:** 개발자는 함수만 제공하고, 언제 어떻게 호출할지는 LLM이 결정한다.

### 함수 정의 방법

```csharp
// 파라미터 없는 단순 함수
string GetCurrentTime()
{
    return DateTime.Now.ToString("yyyy-MM-dd HH:mm:ss");
}

// 파라미터가 있는 함수
string GetWeather(string city)
{
    return $"{city}의 날씨: 맑음, 22도";
}

// 선택적 파라미터
string GetWeatherWithUnit(string city, string unit = "celsius")
{
    return $"{city}: 22{(unit == "fahrenheit" ? "°F" : "°C")}";
}

// 비동기 함수도 가능
async Task<string> GetRealWeather(string city)
{
    var response = await httpClient.GetStringAsync($"...");
    return response;
}
```

### XML 주석의 중요성

LLM이 함수를 올바르게 사용하려면 함수 설명이 필요하다. XML 주석으로 제공한다.

## 4.2 AIAgentBuilder로 Tool 등록

### 기본 Tool 등록

```csharp
// AIAgentBuilder는 Delegate 배열로 Tool을 받는다
// 내부적으로 AIFunctionFactory.Create()를 자동 처리한다

var agent = AIAgentBuilder
    .FromEnvironment()
    .Build(
        name: "WeatherAgent",
        instructions: "사용자의 날씨 관련 질문에 답변한다.",
        tools: new Delegate[] { GetWeather }
    );
```

### 여러 Tool 등록

```csharp
var agent = AIAgentBuilder
    .FromEnvironment()
    .Build(
        name: "MultiToolAgent",
        instructions: "다양한 정보를 제공하는 에이전트다.",
        tools: new Delegate[] { GetWeather, GetCurrentTime, GetExchangeRate }
    );
```

## 4.3 단일 Tool 예제 - 날씨 조회

### 프로젝트 설정

```bash
dotnet new console -o AgentWithTools
cd AgentWithTools
dotnet add package Microsoft.Agents.AI
dotnet add package Microsoft.Agents.AI.OpenAI
dotnet add package DotNetEnv
```

### Program.cs

```csharp
using DotNetEnv;
using AgentFrameworkBook.Shared;

Env.Load();

Console.WriteLine("🌤️ 날씨 조회 Agent");
Console.WriteLine(new string('=', 50));

/// <summary>
/// 지정된 도시의 현재 날씨를 조회한다.
/// </summary>
/// <param name="city">도시 이름 (예: 서울, 부산, 제주)</param>
/// <returns>해당 도시의 날씨 정보</returns>
string GetWeather(string city)
{
    Console.WriteLine($"  [Tool 실행] GetWeather(\"{city}\")");

    // 실제 환경에서는 날씨 API를 호출한다
    var random = new Random();
    var conditions = new[] { "맑음", "흐림", "비", "눈", "구름 조금" };
    var condition = conditions[random.Next(conditions.Length)];
    var temperature = random.Next(-5, 35);

    var result = $"{city}: {condition}, {temperature}도";
    Console.WriteLine($"  [Tool 결과] {result}");
    return result;
}

var agent = AIAgentBuilder
    .FromEnvironment()
    .Build(
        name: "WeatherAgent",
        instructions: @"당신은 날씨 정보를 제공하는 친절한 어시스턴트다.
GetWeather 함수를 사용하여 날씨를 조회하고,
조회한 정보를 자연스럽게 전달한다.
날씨에 따라 적절한 조언도 추가한다.",
        tools: new Delegate[] { GetWeather }
    );

var questions = new[]
{
    "서울 날씨 어때?",
    "부산은 지금 날씨가 어떤가요?",
    "제주도 날씨 알려주세요"
};

foreach (var question in questions)
{
    Console.WriteLine($"\n💬 사용자: {question}\n");
    var result = await agent.RunAsync(question);
    Console.WriteLine($"🤖 Agent: {result.Text}");
    Console.WriteLine(new string('-', 50));
    await Task.Delay(1000);
}

Console.WriteLine("\n✅ 완료!");
```

### 실행 결과

```

==================================================

💬 사용자: 서울 날씨 어때?

  [Tool 실행] GetWeather("서울")
  [Tool 결과] 서울: 맑음, 22도

🤖 Agent: 서울의 현재 날씨는 맑음이고 기온은 22도입니다! 🌤️
          날씨가 좋으니 외출하기 좋은 날입니다.
```

## 4.4 여러 개의 Tools 사용하기

### 다양한 유틸리티 함수 정의

```csharp
/// <summary>지정된 도시의 현재 날씨를 조회한다.</summary>
/// <param name="city">도시 이름</param>
string GetWeather(string city)
{
    Console.WriteLine($"  🌤️ GetWeather(\"{city}\")");
    var random = new Random();
    var conditions = new[] { "맑음", "흐림", "비", "눈" };
    var temp = random.Next(-5, 35);
    return $"{city}: {conditions[random.Next(conditions.Length)]}, {temp}도";
}

/// <summary>지정된 도시의 현재 시간을 반환한다.</summary>
/// <param name="city">도시 이름 (서울, 뉴욕, 런던 등)</param>
string GetCurrentTime(string city)
{
    Console.WriteLine($"  🕐 GetCurrentTime(\"{city}\")");
    var offsets = new Dictionary<string, int>
    {
        { "서울", 9 }, { "뉴욕", -5 }, { "런던", 0 },
        { "도쿄", 9 }, { "파리", 1 }
    };
    var offset = offsets.GetValueOrDefault(city, 9);
    var localTime = DateTime.UtcNow.AddHours(offset);
    return $"{city} 현재 시간: {localTime:HH:mm:ss}";
}

/// <summary>두 도시 사이의 대략적인 거리를 계산한다.</summary>
/// <param name="from">출발 도시</param>
/// <param name="to">도착 도시</param>
string CalculateDistance(string from, string to)
{
    Console.WriteLine($"  📏 CalculateDistance(\"{from}\", \"{to}\")");
    var distances = new Dictionary<string, int>
    {
        { "서울-부산", 325 }, { "서울-제주", 450 },
        { "서울-뉴욕", 11000 }, { "서울-도쿄", 1160 }
    };
    var key = $"{from}-{to}";
    var dist = distances.GetValueOrDefault(key,
               distances.GetValueOrDefault($"{to}-{from}", 1000));
    return $"{from}→{to}: 약 {dist}km";
}

/// <summary>지정된 통화의 현재 환율을 조회한다 (KRW 기준).</summary>
/// <param name="currency">통화 코드 (USD, JPY, EUR 등)</param>
string GetExchangeRate(string currency)
{
    Console.WriteLine($"  💱 GetExchangeRate(\"{currency}\")");
    var rates = new Dictionary<string, double>
    {
        { "USD", 1320.50 }, { "JPY", 9.15 },
        { "EUR", 1450.30 }, { "CNY", 182.40 }
    };
    var rate = rates.GetValueOrDefault(currency.ToUpper(), 0);
    return rate == 0
        ? $"{currency} 환율 정보 없음"
        : $"1 {currency} = {rate:N2} KRW";
}
```

### 멀티 Tool Agent 구성

```csharp
var agent = AIAgentBuilder
    .FromEnvironment()
    .Build(
        name: "MultiToolAgent",
        instructions: @"당신은 다양한 정보를 제공하는 유능한 어시스턴트다.
사용 가능한 도구: GetWeather, GetCurrentTime, CalculateDistance, GetExchangeRate.
질문을 분석하여 적절한 도구를 선택하고, 필요하면 여러 도구를 함께 사용한다.",
        tools: new Delegate[]
        {
            GetWeather, GetCurrentTime, CalculateDistance, GetExchangeRate
        }
    );

// "서울 날씨랑 시간 둘 다 알려줘" 같은 질문에는
// GetWeather, GetCurrentTime 두 Tool이 순차적으로 실행된다
var result = await agent.RunAsync("서울 날씨랑 시간 둘 다 알려줘");
Console.WriteLine(result.Text);
```

## 4.5 외부 API 연동 Tool

### 실제 날씨 API 연동

```csharp
using System.Net.Http.Json;

var httpClient = new HttpClient();
var weatherApiKey = Environment.GetEnvironmentVariable("OPENWEATHER_API_KEY");

/// <summary>
/// OpenWeatherMap API를 사용하여 실제 날씨를 조회한다.
/// </summary>
/// <param name="city">도시 이름 (한글 또는 영어)</param>
async Task<string> GetRealWeather(string city)
{
    // 한글 도시명 → 영어 변환
    var cityMap = new Dictionary<string, string>
    {
        { "서울", "Seoul" }, { "부산", "Busan" }, { "제주", "Jeju" }
    };
    var cityName = cityMap.GetValueOrDefault(city, city);

    var url = $"https://api.openweathermap.org/data/2.5/weather" +
              $"?q={cityName}&appid={weatherApiKey}&units=metric&lang=kr";

    try
    {
        var response = await httpClient.GetFromJsonAsync<WeatherResponse>(url);
        if (response == null) return $"{city} 날씨 정보를 가져올 수 없다.";

        return $"{response.name}: {response.weather[0].description}, " +
               $"{response.main.temp:F1}°C (체감 {response.main.feels_like:F1}°C), " +
               $"습도 {response.main.humidity}%";
    }
    catch (Exception ex)
    {
        return $"날씨 조회 오류: {ex.Message}";
    }
}

// .env에 OPENWEATHER_API_KEY 추가 후 사용
var agent = AIAgentBuilder
    .FromEnvironment()
    .Build("RealWeatherAgent", "실시간 날씨를 제공한다.",
        new Delegate[] { GetRealWeather });
```

### 응답 모델 정의

```csharp
record WeatherResponse(
    string name,
    WeatherMain main,
    WeatherDescription[] weather,
    WindInfo wind
);

record WeatherMain(double temp, double feels_like, int humidity);
record WeatherDescription(string main, string description);
record WindInfo(double speed);
```

### API 호출 캐싱

```csharp
// 간단한 메모리 캐시로 불필요한 API 호출 방지
var cache = new Dictionary<string, (string data, DateTime expires)>();

async Task<string> GetWeatherCached(string city)
{
    if (cache.TryGetValue(city, out var cached) && cached.expires > DateTime.Now)
        return cached.data;  // 캐시 히트

    var data = await GetRealWeather(city);
    cache[city] = (data, DateTime.Now.AddMinutes(5));  // 5분 캐시
    return data;
}
```

## 4.6 Tool 실행 모니터링

### 실행 로그 패턴

Tool이 실행될 때 콘솔에 로그를 남기면 디버깅이 쉬워진다.

```csharp
string GetWeather(string city)
{
    Console.ForegroundColor = ConsoleColor.Yellow;
    Console.WriteLine($"  ▶ [Tool] GetWeather(\"{city}\")");
    Console.ResetColor();

    var result = $"{city}: 맑음, 22도";

    Console.ForegroundColor = ConsoleColor.Green;
    Console.WriteLine($"  ◀ [Result] {result}");
    Console.ResetColor();

    return result;
}
```
