# 5. 대화 흐름 관리

```
 _____ _   _ ____  _____    _    ____  
|_   _| | | |  _ \| ____|  / \  |  _ \ 
  | | | |_| | |_) |  _|   / _ \ | | | |
  | | |  _  |  _ <| |___ / ___ \| |_| |
  |_| |_| |_|_| \_\_____/_/   \_\____/ 
```

지금까지 만든 Agent는 단일 질문에만 답변했다. 실제 대화는 여러 턴에 걸쳐 진행되며, 이전 대화 내용을 기억해야 한다. 이 챕터에서는 Thread를 사용하여 대화 히스토리를 관리하고, 메모리 Provider로 장기 기억을 구현한다.

---

## 5.1 Multi-turn Conversation

### 대화 컨텍스트의 중요성

```
사용자: 서울 날씨 어때?
Agent: 서울은 현재 맑고 22도입니다.

사용자: 거기 비 올까?
Agent: ??? (여기가 어디를 말하는지 모름)
```

Agent가 "거기"가 서울을 의미한다는 것을 이해하려면 이전 대화를 기억해야 한다. 이것이 Thread의 역할이다.

### Thread 개념

Thread는 대화의 문맥을 유지하는 컨테이너다.

```
Thread
├── Message 1: User: "서울 날씨 어때?"
├── Message 2: Agent: "서울은 맑고 22도입니다"
├── Message 3: User: "거기 비 올까?"
└── Message 4: Agent: "서울은 오늘 비 예보가 없습니다"
```

Thread는 모든 대화 메시지, 발화자(사용자/Agent 구분), 시간 순서, Tool 호출 기록을 포함한다.

### Thread로 연속 대화 구현

```csharp
using DotNetEnv;
using AgentFrameworkBook.Shared;

Env.Load();

Console.WriteLine("💬 대화 기억 Agent");
Console.WriteLine(new string('=', 60));

var agent = AIAgentBuilder
    .FromEnvironment()
    .Build(
        name: "MemoryAgent",
        instructions: @"당신은 이전 대화를 기억하는 똑똑한 어시스턴트다.
대화의 맥락을 이해하고 자연스럽게 이어간다.
'그것', '거기', '그때' 같은 대명사를 이전 대화에서 파악하여 이해한다."
    );

// Thread 생성 - 대화 컨텍스트를 유지한다
var thread = agent.CreateThread();

// 연속된 대화 테스트
var conversation = new[]
{
    "내 이름은 김철수야",
    "서울 날씨가 어때?",
    "거기 내일은 어떨까?",
    "내 이름이 뭐였지?"
};

foreach (var message in conversation)
{
    Console.WriteLine($"\n👤 사용자: {message}");

    // Thread를 넘겨서 대화 컨텍스트를 유지한다
    var result = await agent.RunAsync(message, thread);

    Console.WriteLine($"🤖 Agent: {result.Text}");
    Console.WriteLine(new string('-', 60));
    await Task.Delay(1000);
}
```

**실행 결과:**

```
💬 대화 기억 Agent
============================================================

👤 사용자: 내 이름은 김철수야
🤖 Agent: 안녕하세요, 김철수님! 만나서 반갑습니다. 무엇을 도와드릴까요?

------------------------------------------------------------

👤 사용자: 내 이름이 뭐였지?
🤖 Agent: 김철수님이십니다! 😊
```

**핵심 포인트:** `agent.RunAsync(message, thread)`에서 동일한 `thread`를 계속 넘기는 것이 전부다. Agent Framework가 대화 히스토리를 자동으로 관리한다.

## 5.3 대화형 인터페이스 구현

### 실시간 대화 프로그램

```csharp
using DotNetEnv;
using AgentFrameworkBook.Shared;

Env.Load();

Console.WriteLine("💬 대화형 Agent with Thread");
Console.WriteLine(new string('=', 60));
Console.WriteLine("종료: 'quit' | 새 대화: 'reset' | 요약: 'summary'");
Console.WriteLine();

var agent = AIAgentBuilder
    .FromEnvironment()
    .Build(
        name: "ConversationAgent",
        instructions: @"당신은 친근하고 기억력이 좋은 대화 상대다.
사용자가 말한 정보(이름, 취미, 직업 등)를 기억하고 대화에 활용한다."
    );

var thread = agent.CreateThread();
int messageCount = 0;

Console.Write("🤖 Agent: ");
var greet = await agent.RunAsync("간단히 자기소개하고 대화를 시작해줘", thread);
Console.WriteLine(greet.Text);
Console.WriteLine();

while (true)
{
    Console.Write("👤 당신: ");
    var input = Console.ReadLine()?.Trim();

    if (string.IsNullOrWhiteSpace(input)) continue;

    if (input.ToLower() == "quit")
    {
        Console.WriteLine("\n🤖 Agent: 좋은 대화였습니다. 다음에 또 봬요! 👋");
        break;
    }

    if (input.ToLower() == "reset")
    {
        thread = agent.CreateThread();
        messageCount = 0;
        Console.WriteLine("\n✅ 새 대화를 시작한다.\n");
        continue;
    }

    if (input.ToLower() == "summary")
    {
        Console.WriteLine($"\n📊 현재까지 {messageCount}번 대화했다.\n");
        continue;
    }

    Console.Write("\n🤖 Agent: ");
    var result = await agent.RunAsync(input, thread);
    Console.WriteLine(result.Text);
    Console.WriteLine();
    messageCount++;
}
```

## 5.4 Context Provider - 장기 메모리

### 단기 vs 장기 메모리

```
장기 메모리 (Context Provider)       단기 메모리 (Thread)
┌───────────────────────┐           ┌─────────────────────┐
│ • 사용자 이름: 김철수  │           │ • 오늘 대화 내용     │
│ • 직업: 개발자         │     →     │ • 현재 세션 맥락     │
│ • 선호 언어: C#        │           │ • 최근 질문/답변     │
└───────────────────────┘           └─────────────────────┘
        영구 저장                         세션 종료 시 소멸
```

### SimpleMemoryProvider 구현

```csharp
// SimpleMemoryProvider.cs
using System.Text.Json;

public class UserProfile
{
    public string? Name { get; set; }
    public string? Occupation { get; set; }
    public List<string> Interests { get; set; } = new();
    public Dictionary<string, string> Preferences { get; set; } = new();

    public string ToContextString()
    {
        var facts = new List<string>();
        if (!string.IsNullOrEmpty(Name))
            facts.Add($"이름: {Name}");
        if (!string.IsNullOrEmpty(Occupation))
            facts.Add($"직업: {Occupation}");
        if (Interests.Count > 0)
            facts.Add($"관심사: {string.Join(", ", Interests)}");
        return facts.Count > 0
            ? $"[사용자 정보: {string.Join(", ", facts)}]"
            : "";
    }
}

public class SimpleMemoryProvider
{
    private readonly string _filePath;
    private UserProfile _profile = new();

    public SimpleMemoryProvider(string userId)
    {
        var dir = Path.Combine(
            Environment.GetFolderPath(Environment.SpecialFolder.LocalApplicationData),
            "AgentMemory");
        Directory.CreateDirectory(dir);
        _filePath = Path.Combine(dir, $"{userId}.json");
        Load();
    }

    public UserProfile Profile => _profile;

    public void UpdateFromConversation(string conversationSummary)
    {
        // 실제 구현: LLM으로 대화에서 정보를 추출하여 프로필 업데이트
        // 여기서는 간단한 키워드 파싱으로 구현
        if (conversationSummary.Contains("이름은"))
        {
            var start = conversationSummary.IndexOf("이름은") + 3;
            var end = conversationSummary.IndexOf(' ', start);
            if (end > start)
                _profile.Name = conversationSummary[start..end].Trim('이', '다', '야', '.');
        }
        Save();
    }

    private void Load()
    {
        if (File.Exists(_filePath))
        {
            var json = File.ReadAllText(_filePath);
            _profile = JsonSerializer.Deserialize<UserProfile>(json) ?? new();
        }
    }

    public void Save()
    {
        var json = JsonSerializer.Serialize(_profile, new JsonSerializerOptions
        { WriteIndented = true });
        File.WriteAllText(_filePath, json);
    }
}
```

### 메모리를 활용한 Agent

```csharp
using DotNetEnv;
using AgentFrameworkBook.Shared;

Env.Load();

var memory = new SimpleMemoryProvider("user001");
var profileContext = memory.Profile.ToContextString();

var agent = AIAgentBuilder
    .FromEnvironment()
    .Build(
        name: "MemoryAgent",
        instructions: $@"당신은 기억력이 좋은 개인 비서다.
{(string.IsNullOrEmpty(profileContext) ? "" : $"\n사용자 정보:\n{profileContext}\n")}
대화 중 사용자에 대한 새로운 정보를 파악하면 자연스럽게 기억한다."
    );

var thread = agent.CreateThread();

Console.WriteLine("💾 장기 메모리 Agent");
Console.WriteLine(string.IsNullOrEmpty(profileContext)
    ? "  (저장된 사용자 정보 없음)"
    : $"  {profileContext}");
Console.WriteLine();

while (true)
{
    Console.Write("👤 당신: ");
    var input = Console.ReadLine()?.Trim();
    if (string.IsNullOrWhiteSpace(input) || input == "quit") break;

    var result = await agent.RunAsync(input, thread);
    Console.WriteLine($"\n🤖 Agent: {result.Text}\n");
}

// 세션 종료 시 대화 내용 저장
Console.WriteLine("대화 내용을 메모리에 저장하고 있다...");
memory.Save();
```

## 5.5 컨텍스트 윈도우 관리

### 긴 대화에서의 문제

Thread가 너무 길어지면 토큰 비용이 증가하고 처리 시간이 늘어난다. Claude의 컨텍스트 윈도우는 200,000 토큰이지만 비용 최적화를 위해 적절히 관리해야 한다.

```csharp
// 대화가 길어지면 요약해서 새 Thread로 시작한다
async Task<string> SummarizeThread(AIAgent agent, object thread)
{
    var summaryPrompt = "지금까지 나눈 대화의 핵심 내용을 3-5줄로 요약해줘.";
    var result = await agent.RunAsync(summaryPrompt, thread);
    return result.Text;
}

int turnCount = 0;
while (true)
{
    // ... 입력 받기 ...

    var result = await agent.RunAsync(input, thread);
    turnCount++;

    // 20턴마다 Thread 압축
    if (turnCount % 20 == 0)
    {
        Console.WriteLine("📝 대화 요약 중...");
        var summary = await SummarizeThread(agent, thread);

        // 새 Thread에 요약을 첫 메시지로 추가
        thread = agent.CreateThread();
        await agent.RunAsync($"이전 대화 요약: {summary}\n이 맥락을 기억하고 대화를 계속한다.", thread);
        Console.WriteLine("✅ 대화 컨텍스트 압축 완료");
    }
}
```

## 5.6 세션 관리

### 세션 저장과 복원

```csharp
// 대화 히스토리를 JSON 파일로 저장
public class SessionManager
{
    private readonly string _sessionsDir;

    public SessionManager()
    {
        _sessionsDir = Path.Combine(
            Environment.GetFolderPath(Environment.SpecialFolder.LocalApplicationData),
            "AgentSessions");
        Directory.CreateDirectory(_sessionsDir);
    }

    public void SaveSession(string sessionId, List<(string role, string content)> messages)
    {
        var path = Path.Combine(_sessionsDir, $"{sessionId}.json");
        var json = JsonSerializer.Serialize(messages.Select(m => new
        {
            role = m.role,
            content = m.content,
            timestamp = DateTime.Now
        }));
        File.WriteAllText(path, json);
        Console.WriteLine($"💾 세션 저장됨: {sessionId}");
    }

    public string[] ListSessions()
    {
        return Directory.GetFiles(_sessionsDir, "*.json")
            .Select(f => Path.GetFileNameWithoutExtension(f))
            .ToArray();
    }
}
```
