## 6.1 Streaming 응답

### 왜 Streaming인가?

`RunAsync()`는 응답이 완전히 생성된 후에야 반환한다. 긴 응답의 경우 수 초를 기다려야 한다. `RunStreamAsync()`는 생성되는 즉시 청크 단위로 전달하므로 사용자 경험이 크게 개선된다.

```
RunAsync:      [────────────────────] → 전체 응답 한 번에
RunStreamAsync: ▌→ ▌▌ → ▌▌▌ → ...  → 실시간으로 글자 출력
```

### 기본 Streaming 구현

```csharp
using DotNetEnv;
using AgentFrameworkBook.Shared;

Env.Load();

var agent = AIAgentBuilder
    .FromEnvironment()
    .Build("StreamAgent", "친절한 어시스턴트다.");

Console.Write("🤖 Agent: ");

// RunStreamAsync - IAsyncEnumerable<StreamChunk> 반환
await foreach (var chunk in agent.RunStreamAsync("AI 에이전트의 미래에 대해 설명해줘"))
{
    if (!string.IsNullOrEmpty(chunk.Text))
    {
        Console.Write(chunk.Text);  // 실시간 출력
    }
}

Console.WriteLine();
```

### Thread와 함께 Streaming

```csharp
var agent = AIAgentBuilder.FromEnvironment()
    .Build("StreamChatAgent", "대화형 어시스턴트다.");

var thread = agent.CreateThread();

Console.WriteLine("💬 Streaming 대화 Agent (종료: quit)");
Console.WriteLine();

int totalMessages = 0;

while (true)
{
    Console.Write("👤 당신: ");
    var input = Console.ReadLine()?.Trim();

    if (string.IsNullOrWhiteSpace(input)) continue;
    if (input.ToLower() == "quit") break;

    if (input.ToLower() == "stats")
    {
        Console.WriteLine($"\n📊 총 {totalMessages}번 대화했다.\n");
        continue;
    }

    Console.WriteLine();
    Console.Write("🤖 Agent: ");

    try
    {
        await foreach (var chunk in agent.RunStreamAsync(input, thread))
        {
            if (!string.IsNullOrEmpty(chunk.Text))
                Console.Write(chunk.Text);
        }
        totalMessages++;
    }
    catch (Exception ex)
    {
        Console.Write($"\n❌ 오류: {ex.Message}");
    }

    Console.WriteLine("\n");
}
```

### Tool 호출 중 Streaming

Tool을 사용하는 Agent에서 Streaming을 사용하면, Tool 실행 중에는 잠시 멈추고 결과가 나오면 다시 스트리밍이 시작된다.

```csharp
string GetWeather(string city)
{
    Console.WriteLine($"\n  [🌤️ Tool 실행: GetWeather(\"{city}\")]");
    return $"{city}: 맑음, 22도";
}

var agent = AIAgentBuilder.FromEnvironment()
    .Build("WeatherStream", "날씨 에이전트다.",
        new Delegate[] { GetWeather });

Console.Write("🤖 Agent: ");
await foreach (var chunk in agent.RunStreamAsync("서울과 부산 날씨를 비교해줘"))
{
    if (!string.IsNullOrEmpty(chunk.Text))
        Console.Write(chunk.Text);
}
Console.WriteLine();
```

## 6.2 Extended Thinking

### Extended Thinking 개념

Claude가 답변하기 전에 "생각하는" 과정을 거치는 기능이다. 복잡한 추론이 필요한 문제에서 정확도가 크게 향상된다.

```
일반 모드: 질문 → [즉시 답변]

Extended Thinking 모드:
질문 → [생각: 문제 분석 → 접근법 검토 → 단계별 계산] → [최종 답변]
```

**장점:** 복잡한 문제 정확도 향상, 추론 과정 투명성, 오류 감소.  
**단점:** 토큰 비용 증가, 응답 속도 느림.

### Extended Thinking 구현

Extended Thinking은 Agent 옵션을 통해 활성화한다. OpenRouter/Poe를 사용하는 경우 모델이 지원하는지 확인해야 한다.

```csharp
using DotNetEnv;
using Microsoft.Agents.AI;
using Microsoft.Agents.AI.OpenAI;

Env.Load();

var apiKey  = Environment.GetEnvironmentVariable("LLM_API_KEY")!;
var baseUrl = Environment.GetEnvironmentVariable("LLM_BASE_URL") ?? "https://openrouter.ai/api/v1";
var model   = Environment.GetEnvironmentVariable("LLM_MODEL")   ?? "anthropic/claude-sonnet-4-5";

var client = new OpenAIAgentClient(new Uri(baseUrl), apiKey);

// Extended Thinking 활성화
var agent = client.CreateAIAgent(
    model: model,
    name: "ThinkingAgent",
    instructions: "복잡한 문제를 단계별로 신중하게 분석한다.",
    defaultOptions: new Dictionary<string, object>
    {
        { "max_tokens", 16000 },
        { "thinking", new Dictionary<string, object>
            {
                { "type", "enabled" },
                { "budget_tokens", 8000 }
            }
        }
    }
);

var problems = new[]
{
    "125 × 847을 계산하고, 결과가 100,000보다 큰지 판단해줘",
    "피보나치 수열의 10번째 항은? 계산 과정도 설명해줘"
};

foreach (var problem in problems)
{
    Console.WriteLine($"❓ 문제: {problem}\n");

    bool inThinking = false;

    await foreach (var chunk in agent.RunStreamAsync(problem))
    {
        if (chunk.Type is "thinking" or "text_reasoning")
        {
            if (!inThinking)
            {
                Console.ForegroundColor = ConsoleColor.DarkGray;
                Console.Write("[🧠 추론] ");
                inThinking = true;
            }
            Console.Write(chunk.Text);
        }
        else if (!string.IsNullOrEmpty(chunk.Text))
        {
            if (inThinking)
            {
                Console.ResetColor();
                Console.WriteLine("\n");
                Console.Write("[💬 답변] ");
                inThinking = false;
            }
            Console.Write(chunk.Text);
        }
    }

    Console.ResetColor();
    Console.WriteLine("\n" + new string('-', 60) + "\n");
    await Task.Delay(2000);
}
```

### Thinking 예산 관리

```csharp
// 문제 복잡도에 따라 예산 조정
int GetThinkingBudget(string problem)
{
    var complexity = problem.Length + problem.Count(c => c == '?' || c == '이');
    return complexity > 50 ? 8000 : 2000;
}

var budget = GetThinkingBudget(problem);
// defaultOptions의 budget_tokens를 동적으로 설정
```

**언제 사용할까?** 수학 계산, 다단계 추론, 코드 디버깅, 전략적 의사결정에 적합하다. 간단한 질답에는 비용 낭비이므로 비활성화한다.

## 6.3 커스텀 웹 검색 Tool

Hosted Web Search Tool은 Poe/OpenRouter 환경에서 지원되지 않을 수 있다. 직접 구현한다.

```csharp
var httpClient = new HttpClient();

/// <summary>
/// DuckDuckGo 즉석 검색 (API 키 불필요)로 웹 정보를 검색한다.
/// </summary>
/// <param name="query">검색할 키워드 또는 질문</param>
async Task<string> SearchWeb(string query)
{
    Console.WriteLine($"\n  [🔍 검색: \"{query}\"]");
    try
    {
        // DuckDuckGo 즉석 답변 API (무료, API 키 불필요)
        var url = $"https://api.duckduckgo.com/?q={Uri.EscapeDataString(query)}&format=json&no_redirect=1";
        var json = await httpClient.GetStringAsync(url);
        var doc = System.Text.Json.JsonDocument.Parse(json);

        var abstract_ = doc.RootElement.TryGetProperty("Abstract", out var abs)
            ? abs.GetString() ?? "" : "";
        var answer = doc.RootElement.TryGetProperty("Answer", out var ans)
            ? ans.GetString() ?? "" : "";

        if (!string.IsNullOrEmpty(abstract_))
            return $"검색 결과: {abstract_}";
        if (!string.IsNullOrEmpty(answer))
            return $"검색 결과: {answer}";

        return $"'{query}'에 대한 즉석 결과가 없습니다. 일반 지식으로 답변합니다.";
    }
    catch (Exception ex)
    {
        return $"검색 오류: {ex.Message}";
    }
}

var agent = AIAgentBuilder.FromEnvironment()
    .Build("SearchAgent",
        "최신 정보가 필요한 질문은 SearchWeb 도구를 사용한다.",
        new Delegate[] { SearchWeb });
```

## 6.4 Middleware로 Agent 확장

### Middleware 개념

Middleware는 Agent 요청/응답 사이에 처리를 추가하는 패턴이다.

```
사용자 요청 → [로깅] → [필터링] → Agent → [후처리] → [로깅] → 응답
```

### 로깅 Wrapper 구현

```csharp
// Agent를 감싸는 로깅 래퍼
public class LoggingAgentWrapper
{
    private readonly AIAgent _agent;
    private readonly string _logFile;
    private int _callCount = 0;

    public LoggingAgentWrapper(AIAgent agent, string logDir = "Logs")
    {
        _agent = agent;
        Directory.CreateDirectory(logDir);
        _logFile = Path.Combine(logDir, $"agent_{DateTime.Now:yyyyMMdd_HHmmss}.log");
        Log($"=== Agent Session Started: {DateTime.Now} ===");
    }

    public async Task<string> RunAsync(string input, object? thread = null)
    {
        _callCount++;
        Log($"[#{_callCount}] USER: {input}");

        var sw = System.Diagnostics.Stopwatch.StartNew();
        string response = "";

        try
        {
            RunResult result;
            if (thread != null)
                result = await _agent.RunAsync(input, thread);
            else
                result = await _agent.RunAsync(input);

            response = result.Text;
            sw.Stop();

            Log($"[#{_callCount}] AGENT ({sw.ElapsedMilliseconds}ms): {response}");
            return response;
        }
        catch (Exception ex)
        {
            sw.Stop();
            Log($"[#{_callCount}] ERROR: {ex.Message}");
            throw;
        }
    }

    private void Log(string message)
    {
        var entry = $"[{DateTime.Now:HH:mm:ss}] {message}";
        File.AppendAllText(_logFile, entry + "\n");
        Console.ForegroundColor = ConsoleColor.DarkGray;
        Console.WriteLine($"  LOG: {message[..Math.Min(80, message.Length)]}");
        Console.ResetColor();
    }
}
```

### 콘텐츠 필터 Wrapper

```csharp
public class FilteredAgentWrapper
{
    private readonly AIAgent _agent;
    private readonly string[] _blockedWords;

    public FilteredAgentWrapper(AIAgent agent, string[] blockedWords)
    {
        _agent = agent;
        _blockedWords = blockedWords;
    }

    public async Task<string> RunAsync(string input)
    {
        // 입력 필터
        foreach (var word in _blockedWords)
        {
            if (input.Contains(word, StringComparison.OrdinalIgnoreCase))
                return "⚠️ 이 주제에 대해서는 답변할 수 없습니다.";
        }

        var result = await _agent.RunAsync(input);
        return result.Text;
    }
}
```

### 성능 측정 Wrapper

```csharp
public class MetricsWrapper
{
    private readonly AIAgent _agent;
    private List<(string query, long ms, int chars)> _metrics = new();

    public MetricsWrapper(AIAgent agent) => _agent = agent;

    public async Task<string> RunAsync(string input)
    {
        var sw = System.Diagnostics.Stopwatch.StartNew();
        var result = await _agent.RunAsync(input);
        sw.Stop();

        _metrics.Add((input[..Math.Min(30, input.Length)], sw.ElapsedMilliseconds, result.Text.Length));
        return result.Text;
    }

    public void PrintStats()
    {
        Console.WriteLine($"\n📊 성능 통계 ({_metrics.Count}회 호출):");
        Console.WriteLine($"  평균 응답 시간: {_metrics.Average(m => m.ms):F0}ms");
        Console.WriteLine($"  최대 응답 시간: {_metrics.Max(m => m.ms)}ms");
        Console.WriteLine($"  평균 응답 길이: {_metrics.Average(m => m.chars):F0}자");
    }
}
```
